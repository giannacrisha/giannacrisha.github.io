## How to Use These Icons

### Option 1: Direct SVG Embed
Copy the SVG code directly into your HTML:
```html
<svg id="Heart" xmlns="http://www.w3.org/2000/svg" viewBox="0 0 24 24">
  <!-- SVG content here -->
</svg>
```

### Option 2: SVG File
Save the SVG code to a file and reference it:
```html
<img src="heart.svg" alt="Heart" />
```

### Option 3: NPM Package
```bash
npm install @hackernoon/pixel-icon-library
```

Use in your project:
```html
<i class="hn hn-heart"></i>
<i class="hn hn-heart-solid"></i>
```

### Option 4: GitHub Direct Link
```html
<img src="https://raw.githubusercontent.com/hackernoon/pixel-icon-library/main/icons/SVG/regular/heart.svg" alt="Heart" />
```

## Missing Icons (Custom Required)

The following icons are **not available** in the library and require custom creation:
1. **Eraser** - No dedicated eraser icon (Broom available as alternative)
2. **Paint Bucket/Fill** - No dedicated fill tool
3. **Eyedropper/Color Picker** - Eye icon available but not exact match
4. **Line Drawing Tool** - No dedicated line tool
5. **Rectangle/Square Drawing** - No dedicated rectangle tool
6. **Zoom In/Out** - No dedicated zoom icons (Search available)
7. **Move/4-Way Arrow** - Individual arrows available, not 4-way combined
