# obsidian templates for gi's garden

copy all files from this folder into:
  src/content/templates/

then in obsidian:
  cmd+p → insert template → pick the one you need

---

| template     | use for                                           |
|--------------|---------------------------------------------------|
| case-study   | design projects: problem/research/solution/impact |
| essay        | longer prose, arguments, reflections              |
| poem         | poems, verse, lyric writing                       |
| thought      | short observations, fragments, questions          |
| lab-project  | code projects, experiments, builds                |
| gallery      | artwork, illustrations, sketches                  |
| library      | films, books, albums, shows                       |

---

file naming rules when saving from obsidian to the astro project:
- all lowercase
- dashes not spaces: my-poem-title.mdx
- change extension from .md to .mdx
- no special characters in the filename
